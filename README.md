# üêæ Purrfect Match

**Purrfect Match** is a mobile application designed to help users adopt cats from rescue shelters through a **Tinder-like swiping experience**. The app makes pet adoption more engaging and accessible by pairing intuitive UI/UX design with real-time backend services.

Built as an Android application with a strong emphasis on **user experience, interaction design, and simplicity**.

---

## ‚ú® Features

* **Swipe-Based Matching**

  * Tinder-style left/right swipe interface
  * Like or pass on cats available for adoption

* **Cat Profiles**

  * Photos, names, age, breed, and personality traits
  * Clear, friendly presentation to encourage adoption

* **User Accounts**

  * Secure authentication and session management
  * Personalized adoption preferences

* **Real-Time Data**

  * Cat listings and user interactions stored in Firebase
  * Instant updates across devices
